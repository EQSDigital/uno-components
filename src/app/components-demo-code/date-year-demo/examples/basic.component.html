You can config the number of years before, and after, current date, that fill the dropdown list - @Input "numYearsBefore" and/or "numYearsAfter".

A default number is provided for both, if none/any of the Inputs are passed in.

Component's Input Directive "year" is only needed if you want to control the <uno-date-year>'s showing year from the outside of the component.
